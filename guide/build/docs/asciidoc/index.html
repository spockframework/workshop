2.1. Scenario

Your team is tasked to provide an implementation of FixedLengthIntStack. Your co-worker already began yesterday, but got sick today, luckily he committed his changes before he left.

In src/main/java/step1/FixedLengthIntStack.java you’ll find the skeleton of the implementation. In src/test/groovy/step1/FixedLengthIntStackSpec.groovy you’ll find a Specification with the description of what should be happening.

Your task is to turn the specification description into code, and then fix the implementation. In true TDD fashion you’ll write the test first, and then fix the implementation until all tests are green.
